webpack配置导入css

本文最后更新于：几秒前

最简单的 CSS 配置

  1. 安装依赖：cnpm i --save-dev css-loader style-loader

  2. 在 webpack.config.js 中添加规则：

    module: {
 rules: [
     {
         test: /\.css$/,
         use: [
             {
                 loader: 'style-loader'  // 可以把css放在页面上
             },
             {
                 loader: 'css-loader'    // 放在后面的先被解析
             }
         ]
     }

    ]
    
}
